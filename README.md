# JYC Portal

An AI-powered college youth club portal built for Jaypee University students.

## Features

- Student authentication system
- Event registrations
- Photo gallery
- Results section
- Feedback system
- Club management features
- MongoDB database integration

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
